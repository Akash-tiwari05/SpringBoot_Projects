# Spring Boot Learning Projects: Tax Application & OTT Subscription

Welcome to my **Spring Boot Learning Projects** repository! This collection of projects is designed to help students like me dive into the world of **Spring Boot** through practical examples. In this repository, I will work on two key projects that will help me build essential skills:

1. **Tax Application**: A simple application that calculates taxes based on different income slabs.
2. **OTT Subscription Management**: A system that manages subscriptions for an Over-the-Top (OTT) media service.

These projects will teach me how to develop real-world applications using **Spring Boot** and help me grasp key concepts such as REST APIs, databases, and authentication.


## Introduction

By completing these projects, I will learn how to design and implement services, manage databases, secure applications, and build REST APIs.

## Tech Stack

Here are the main technologies I'll use for these projects:

- **Spring Boot**: The main framework to build Java-based applications.
- **Spring Data JPA**: For interacting with databases.
- **Spring Security**: For securing the application and handling user authentication.
- **H2 Database** (or MySQL/PostgreSQL): For storing data (H2 is used for simplicity).
- **Thymeleaf** (optional): If I'm building a web front-end.
- **REST APIs**: To interact with the backend.
- **JUnit**: For writing tests.

## Learning Outcomes

By the end of these projects, I will:
- Gain hands-on experience in building RESTful services with Spring Boot.
- Understand how to handle user authentication and authorization using **Spring Security**.
- Learn to interact with databases using **Spring Data JPA**.
- Understand how to manage different features in a project like tax calculation and subscription management.
- Learn to deploy and run Spring Boot applications locally.
